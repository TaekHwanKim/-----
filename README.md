# PortFolio

### 파이썬 프로젝트, 공부 코딩
  - tkinter를 이용한 GUI 프로그래밍
    - 버튼, 레이블, 리스트박스 등의 객체 생성과 이벤트 발생에 대해 공부
  - Programmers 사이트의 코딩테스트 문제풀이
    - 각종 자료구조와 알고리즘으로 문제를 해결
  - Tic_Tac_Toe 게임 프로젝트
    - 텍스트로 구현한 틱택토 게임
    - tkinter를 이용하여 GUI로 구현한 틱택토 게임
  - selenium을 이용한 자동화 크롤링
    - 구글 검색과 이미지 다운로드를 코딩으로 자동화하여 데이터를 크롤링
